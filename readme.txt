Treinaweb Anúncios allows you to add image ads in your website or blog, in a simple way.

== Description ==

Created on the course: "Desenvolvimento de Plugins para Wordpres" of the online school TreinaWeb, this plugin allows the user to create and show ads in two ways: widget or shortcode. The plugin also provides a dashboard and displays some reports.

### Descrição em Português: ###

Criado no curso de Wordpress desenvolvimento de Pugins do Treinaweb Cursos. Esse plugin permite ao usuário cadastrar anúncios e exibi-los utilizando Widget ou Shortcode. O plugin disponibiliza um Dashboard Widget com relatório de exibição de todos os anúncios.


== Installation ==

Upload the Treinaweb Anúncios plugin to your blog and Activate it.
